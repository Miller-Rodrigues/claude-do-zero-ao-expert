Sobre este site

Claude do Zero ao Expert é um site educacional independente, feito para guiar um iniciante completo em IA até o nível avançado no uso das soluções Claude: Chat, Cowork, Code e Design.

Cada curso é dividido em 24 módulos, do fundamental ao avançado, com passo a passo, casos de uso reais e vídeos selecionados. Use o menu lateral de cada curso para navegar entre os módulos na ordem que preferir.

Este site não é afiliado oficialmente à Anthropic; consulte sempre a documentação oficial para informações atualizadas sobre os produtos.
